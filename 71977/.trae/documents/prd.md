## 1. 产品概述

趣味太阳系模拟器——一款基于浏览器的交互式天文模拟游戏，以俯视视角呈现太阳系行星公转运动。玩家可以自由缩放、旋转视角，点击行星查看真实天文数据，并通过时间加速/减速/暂停功能探索宇宙运行的奥秘。

- 目标用户：天文爱好者、学生群体、科普教育场景
- 核心价值：以趣味互动方式直观理解太阳系行星运动规律

## 2. 核心功能

### 2.1 功能模块

1. **太阳系模拟主画面**：Canvas 渲染太阳、八大行星、轨道线
2. **行星信息面板**：点击行星弹出详细信息卡片
3. **时间控制面板**：加速/减速/暂停/重置时间流速
4. **视角操控**：鼠标滚轮缩放、拖拽旋转视角

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主页面 | 太阳系模拟画布 | Canvas 全屏渲染太阳系俯视图，行星按真实比例速度公转，显示轨道线 |
| 主页面 | 行星信息面板 | 点击行星后侧边滑出面板，显示名称、直径、温度、公转周期、距日距离等 |
| 主页面 | 时间控制栏 | 底部工具栏，包含暂停/播放、加速(×2 ×5 ×10)、减速(×0.5)、重置按钮及当前倍速显示 |
| 主页面 | 视角控制 | 鼠标滚轮缩放、拖拽平移画布，双击行星居中聚焦 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空黑（#0a0e1a）+ 星云蓝紫渐变点缀
- **强调色**：星光金（#f0c040）用于太阳和高亮交互
- **按钮风格**：毛玻璃质感圆角按钮，半透明背景 + 发光边框
- **字体**：Orbitron（显示字体，太空科幻感）+ Noto Sans SC（中文正文）
- **布局**：全屏 Canvas + 浮动面板叠加
- **图标**：线性图标风格，配合科技感 UI

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主页面 | 太阳系画布 | 全屏深色背景，太阳中心发光渐变，行星彩色圆点带光晕，轨道虚线 |
| 主页面 | 行星信息面板 | 右侧滑出卡片，毛玻璃背景，行星名称大字+数据列表，关闭按钮 |
| 主页面 | 时间控制栏 | 底部居中浮动栏，暂停/播放图标按钮，倍速按钮组，当前倍速数字显示 |
| 主页面 | 缩放提示 | 左下角小字提示操作方式 |

### 4.3 响应式

- 桌面优先设计，Canvas 自适应窗口大小
- 面板在小屏幕下全宽展示
- 触屏设备支持双指缩放和拖拽

### 4.4 3D 场景指引

- 本项目为 2D 俯视视角，使用 Canvas 2D API 渲染
- 太阳：中心发光渐变圆 + 脉动光晕动画
- 行星：彩色圆点 + 外发光效果，大小按对数比例缩放
- 轨道：半透明虚线圆环
- 背景：深空色 + 随机星点粒子
