## 1. 产品概述（v2.0 大型迭代）

接鸡蛋农场游戏是一款趣味休闲网页游戏——玩家在屏幕底部操控篮子接住母鸡下落的鸡蛋来得分，同时躲避坏蛋和炸弹。v2.0 版本将扩展为包含多种掉落物、道具系统、农场升级、多关卡、成就收集、双人协作的完整休闲游戏。

**核心价值**：简单上手的操控 + 丰富的成长系统 + 多样的玩法模式 = 长久留存的休闲体验

## 2. 核心功能（v2.0）

### 2.1 功能模块

1. **多种掉落物系统**
   - 普通蛋 🥚：+10分
   - 金蛋 🪙：+50分（稀有）
   - 彩蛋 🎁：随机道具
   - 小鸡 🐥：+25分（特殊奖励）
   - 粪便 💩：-20分
   - 炸弹 💣：-1生命
   - 爱心 ❤️：+1生命

2. **道具系统**（接住彩蛋触发）
   - ⚡ 加速移动：篮子移动速度提升50%，持续8秒
   - 📏 加宽篮子：篮子宽度增加50%，持续10秒
   - 🧲 磁力吸附：鸡蛋自动被吸引，持续6秒
   - 🛡️ 无敌护盾：免疫负面效果，持续5秒
   - ⏸️ 时间减速：所有掉落物减速，持续7秒

3. **农场升级系统**
   - 等级1：基础鸡窝，1只母鸡，仅普通蛋/炸弹/粪便
   - 等级2：升级鸡窝，下蛋频率+15%，解锁金蛋（5%）
   - 等级3：豪华鸡窝，2只母鸡，解锁彩蛋（3%）
   - 等级4：黄金鸡窝，下蛋频率+30%，解锁小鸡（2%）
   - 等级5：传说鸡窝，3只母鸡，解锁爱心道具

4. **多关卡场景**
   - 关卡1：春日农场（晴朗，基础掉落）
   - 关卡2：夏日牧场（微风，轻微左右飘移）
   - 关卡3：秋日果园（落叶，金蛋概率+2%）
   - 关卡4：冬日雪原（下雪，掉落速度-20%）
   - 关卡5：雷电农场（闪电，随机加速掉落）

5. **成就收集系统**
   - 鸡蛋图鉴：收集10种不同鸡蛋/道具
   - 里程碑成就：接蛋100/500/1000个、单局1000分等
   - 收藏展示：已解锁成就徽章墙

6. **双人协作模式**
   - 双人同屏：左侧玩家（WASD）+ 右侧玩家（方向键）
   - 颜色匹配：蓝篮子接蓝蛋、红篮子接红蛋
   - 协作计分：共同分数，共同生命
   - 配合挑战：特殊组合蛋双倍分

7. **排行榜和每日任务**
   - 本地排行榜：历史最高分、单局最多接蛋等
   - 每日任务：每日3个挑战任务，完成奖励
   - 数据统计：总接蛋数、总游戏时长、最佳单局

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主菜单 | 导航区 | 单人游戏、双人模式、升级农场、成就图鉴、每日任务 |
| 游戏主界面 | 状态栏 | 分数、生命、倒计时、当前道具、农场等级 |
| 游戏主界面 | 游戏画布 | 母鸡移动、多种掉落物、篮子、天气特效 |
| 升级界面 | 升级面板 | 农场等级展示、升级花费、属性预览 |
| 成就界面 | 图鉴墙 | 收集进度、成就徽章、解锁条件 |
| 任务界面 | 任务列表 | 每日3任务、完成状态、奖励领取 |
| 游戏结束 | 结果展示 | 最终得分、本局成就、历史对比 |

## 4. 用户界面设计

### 4.1 设计风格（延续并增强）

- **主色调**：温暖的农场绿（#4A7C3F）与麦田金（#F5C542）
- **新增主题色**：
  - 道具蓝：#3498DB
  - 魔法紫：#9B59B6
  - 火焰橙：#E67E22
- **按钮风格**：圆角立体按钮，木纹质感，按材质区分功能类型
- **字体**：Fredoka One（标题/数字）+ Nunito（正文）
- **新增UI元素**：道具栏、升级面板、图鉴格子、任务卡片

### 4.2 新增特效

- 道具激活时全屏闪光+图标显示
- 金蛋/小鸡接住时庆祝粒子效果
- 天气系统动画（雨滴、雪花、落叶、闪电）
- 磁力吸附牵引光线
- 护盾发光边框
- 升级成功烟花效果

### 4.3 响应式设计

- 双人模式仅在屏幕宽度≥1024px可用
- 移动端简化为单人模式+触控优化
- 升级/成就面板在移动端全屏展示
