# 空中跳伞 v2.0 版本升级文档

## 升级概述

从 2D Canvas 版本升级为全 3D 沉浸式跳伞体验，引入 Three.js 3D 引擎，支持多种场景、天气系统、多人竞技和特技动作。

## 新增功能清单

### 1. 3D 第一人称视角
- 支持第一人称（FPV）和第三人称（TPV）视角切换
- 真实的 3D 自由落体物理模拟
- 头部跟踪和视角旋转
- 360° 全景视角支持

### 2. 多种跳伞场景
| 场景名称 | 环境特点 | 难度等级 |
|---------|---------|---------|
| 🏔️ 山脉 | 雪山、峡谷、森林 | ⭐⭐ |
| 🏖️ 海滩 | 海岸线、岛屿、游轮 | ⭐ |
| 🏙️ 城市天际线 | 摩天大楼、直升机坪 | ⭐⭐⭐ |

### 3. 角色定制系统
- **降落伞皮肤**: 经典红、迷彩、荧光、彩虹、国旗系列
- **角色装备**: 头盔、护目镜、飞行服、手套
- **解锁系统**: 通过游戏币解锁新装备

### 4. 天气系统
- **风力偏移**: 影响水平移动速度和方向
- **云层遮挡**: 降低能见度，增加难度
- **雷暴区域**: 危险区域，需要躲避
- **动态天气**: 晴朗 → 多云 → 雷雨

### 5. 多人竞技模式
- 最多 4 人同时跳伞
- 编队飞行（Formation Skydiving）
- 实时位置显示
- 排行榜系统

### 6. 特技动作系统
| 特技 | 操作方式 | 分数加成 |
|-----|---------|---------|
| 前空翻 | 快速连续按 ↑↑ | +200 |
| 后空翻 | 快速连续按 ↓↓ | +200 |
| 横滚 | 快速连续按 ←← 或 →→ | +150 |
| 俯冲 | 长按 ↓ | +50/秒 |
| 滑翔 | 长按 ↑ | +30/秒 |

### 7. VR 支持
- 头戴设备视角跟踪
- 体感控制支持
- 360° 全景视频录制

## 技术架构升级

### 技术栈变更
- **渲染引擎**: Three.js r160
- **物理引擎**: Cannon.js
- **3D 模型**: GLTF 格式
- **音效**: Web Audio API
