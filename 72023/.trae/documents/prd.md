## 1. 产品概述

海绵宝宝海底冒险是一款基于 HTML5 Canvas 的 2D 横版冒险游戏。玩家控制海绵宝宝在多样化的海底场景中自由游动，收集珍珠和金币，躲避不同 AI 类型的海底生物攻击，补充氧气维持生命，探索隐藏洞穴，挑战 Boss，通过收集足够宝物来过关。支持单人冒险和双人协作两种游戏模式。

- 目标用户：休闲游戏玩家、海绵宝宝爱好者、多人协作游戏玩家
- 核心价值：趣味性的海底冒险体验，多样化的地图探索，策略性的角色能力搭配

## 2. 核心功能

### 2.1 用户角色

| 角色 | 操作方式 | 核心能力 |
|------|----------|----------|
| 海绵宝宝 | 方向键控制（P1：WASD，P2：方向键） | 上下左右自由游动 |
| 派大星 | 可解锁角色 | 防御力提升，碰撞掉血减半 |
| 章鱼哥 | 可解锁角色 | 氧气消耗降低 30% |
| 蟹老板 | 可解锁角色 | 收集金币分数翻倍 |
| 珊迪 | 可解锁角色 | 移动速度提升 25% |

### 2.2 功能模块

1. **主菜单界面**：游戏标题、开始按钮、模式选择（单人/双人）、角色选择、地图选择
2. **角色选择界面**：可解锁角色展示、特殊能力说明
3. **地图选择界面**：4 张海底地图预览、难度说明
4. **游戏主界面**：海底场景、角色控制、宝物收集、敌人躲避、氧气系统、隐藏洞穴
5. **Boss 战界面**：Boss 血条、特殊攻击模式、掉落奖励
6. **过关/结束界面**：过关提示、最终得分、解锁提示

## 3. 核心流程

玩家进入主菜单 → 选择单人/双人模式 → 选择角色 → 选择地图 → 进入关卡 → 控制角色游动 → 收集宝物 → 补充氧气罐 → 躲避不同AI的敌人 → 探索隐藏洞穴 → 收集达标后触发 Boss 战 → 击败 Boss 过关 → 解锁新角色/地图 → 进入下一关 → 生命耗尽或氧气耗尽则游戏结束

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深海蓝(#0a1628) + 海洋青(#00e5ff) + 珊瑚橙(#ff6b35)
- 地图主题色：
  - 珊瑚礁：翠绿(#2ecc71) + 粉红(#ff6b9d)
  - 深海沟：深紫(#2c1654) + 幽蓝(#1a237e)
  - 沉船区：铁锈棕(#8b4513) + 腐绿(#556b2f)
  - 火山口：熔岩红(#d32f2f) + 暗橙(#ff5722)
- 辅助色：沙黄(#ffd93d)、珍珠白(#f0f8ff)、海藻绿(#2ecc71)
- 按钮风格：圆润3D按钮，带光泽和阴影，悬浮放大动效
- 字体：标题使用 Fredoka One 风格的圆润字体，正文使用 Nunito
- 布局风格：全屏 Canvas 游戏，顶部 HUD 叠加层，左侧生命氧气，右侧得分进度
- 图标/表情：海洋主题手绘风格，圆润可爱

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主菜单 | 标题区 | 超大圆润字体，渐变色，气泡上升动画，鱼群游动背景 |
| 主菜单 | 模式切换 | 双人模式：两个角色并排显示，P1/P2 标签 |
| 角色选择 | 卡片列表 | 5 张角色卡片，头像、名称、特殊能力图标，选中高亮边框 |
| 地图选择 | 缩略图网格 | 2x2 地图缩略图，地图名称，难度星标，锁定/解锁状态 |
| 游戏主界面 | 海底背景 | 4种主题：珊瑚礁（多彩珊瑚）、深海沟（发光生物）、沉船区（破旧船体）、火山口（熔岩裂缝） |
| 游戏主界面 | 光影特效 | 上方光束折射、水中悬浮物、上升气泡、成群小鱼、水波纹 |
| 游戏主界面 | 氧气条 | 左侧垂直蓝色渐变条，低于20%时红色闪烁 |
| 游戏主界面 | 氧气罐 | 蓝色发光气瓶，可拾取补充 30% 氧气 |
| 游戏主界面 | 隐藏洞穴 | 暗色岩石拱门，入口有神秘光芒，进入后场景切换 |
| Boss战 | Boss血条 | 顶部居中 Boss 名称 + 红色血条 |
| 双人模式 | 双HUD | 左右两侧分别显示 P1 和 P2 的生命 |
| 游戏主界面 | HUD | 左上生命心形、左中氧气条、顶部中间收集进度、右上分数、右下地图名称 |

### 4.3 响应式设计

- 桌面优先设计，Canvas 自适应窗口大小
- 双人模式仅支持键盘操作
- 游戏画面保持 16:9 比例，居中显示

### 4.4 3D场景指引

不适用，本项目为2D Canvas游戏。
